Datenschutz & Sicherheit
Deine Fragen werden verschlüsselt an die Anthropic Claude API übermittelt, um eine Antwort zu generieren.
Zur semantischen Suche erstellt die OpenAI Embeddings API Vektoren aus deinen Fragen – keine Inhalte werden gespeichert.
Deine Dokumente liegen verschlüsselt in einer Supabase-Datenbank (EU-Region) und werden nicht an Dritte weitergegeben.
Es werden keine Gesprächsinhalte dauerhaft auf Servern gespeichert. Der Chatverlauf verbleibt ausschließlich in deinem Browser.
KLEO verarbeitet ausschließlich die Dokumente, die du selbst hochlädst. Keine Weitergabe an Dritte. Keine Werbung.
